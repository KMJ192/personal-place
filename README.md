# REACT-FUNCTION

react에서 사용할 수 있는 custom hooks 또는 component, utile함수들을 개발하기 위한 repository입니다.

### dev stack

- React
- NextJS
- typescript
- vite
- yarn berry zero install

### 프로젝트 실행

```
yarn start
```

### custom hooks

##### useComponentDidMount

- 컴포넌트가 mount됐을 때 한번 실행

##### useDebounce

- debounce hook

##### useThrottle

- throttle hook

##### useTimeout

- timeout hook

##### useInterval

- interval hook

##### useIntersectionObserver

- intersection observer hooks

##### useTrie

- trie hooks
